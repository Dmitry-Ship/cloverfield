# Cloverfield


## Setup

```
  git clone https://github.com/Dmitry-Ship/cloverfield.git
  cd cloverfield
  npm install
```
**Poduction-ready build with minification**
```
  npm start
```
**Development build with react-hot-loader**
```
  npm run start:dev
```

## Alternative setup with Yarn

```
  git clone https://github.com/Dmitry-Ship/cloverfield.git
  cd cloverfield
  yarn install
```
**Poduction-ready build with minification**
```
  yarn start
```
**Development build with react-hot-loader**
```
  yarn start:dev
```
